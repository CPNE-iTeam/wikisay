# Wikisay
A cowsay like cli command, but it returns a wiki answer

</div>

### Built with
* [![Nodejs][Node.js]][Node-url]




### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```


## Installation

Install Wikisay with
```sh
sudo npm install -g wikisay
```

## Usage
Run it with 
```sh
wikisay
```
